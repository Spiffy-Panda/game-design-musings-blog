[Lo]gical [Ma]gic

An invented magic system built on first- and second-order logic — ∀ is a ward, ∃ is a wish, a proof is a spell, and every miracle is audited to the stroke. Field-equation-free, calculation-heavy: the metamathematical sibling of Thaumodynamics.

With costs, derivations, and a tuning table.
